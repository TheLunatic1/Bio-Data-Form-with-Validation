# Bio-Data-Form-with-Validation

### Methodology

The system was developed using PHP 8.1+ and runs on XAMPP/WAMP local server.

#### PHP Installation Process:
1. Downloaded XAMPP from https://www.apachefriends.org
2. Installed XAMPP (includes Apache, MySQL, PHP)
3. Started Apache server from XAMPP Control Panel
4. Placed project files in `htdocs/bio-data-form/` folder
5. Accessed via browser: http://localhost/bio-data-form/index.php

#### Technologies Used:
- HTML5, CSS, JavaScript (Client-side validation)
- PHP (Server-side processing & validation)
- POST method for secure form submission
